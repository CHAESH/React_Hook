# @chae_hook/use-title

React Hook to update your document's title.

## Installation

### yarn

```
yarn i @chae_hook/use-title
```

### npm

```
npm i @chae_hook/use-title
```

## Usage

```
import React from "react;
import useTitle from "@chae_hook/use-title;

funciton App() {
    useTitle("Welcome);
    return <h1> Welcome</h1>
}
```
